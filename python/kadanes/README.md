# `kadanes`

Kadane's Algorithm Implemented in Python

## Running App

### Install App

```sh
virtualenv -p python3 venv
```

```sh
source .venv/bin/activate
```

```sh
pip install -r requirements.txt
```

### Running `kadanes`

```sh
python3 -m kadanes
```
